Models

  • Neural networks: These models can be used to learn non-linear relationships between input and output variables. Recurrent Neural Networks (RNN) and Long Short-term Memory (LSTM) networks are particularly useful for modeling time series data.
  • Support Vector Machines (SVMs): These models can be used to learn non-linear decision boundaries in order to classify or predict time series data.
  • Decision Trees and Random Forests: These models can be used to learn non-linear relationships between input and output variables by building a tree-based model.
  • Nonlinear Autoregressive with Exogenous Inputs (NARX) and Nonlinear State Space (NSS) models: These models can be used to model non-linear dynamic systems using a combination of autoregressive and exogenous inputs.
  • Nonlinear autoregressive moving average with exogenous inputs (NARMAX) model: This model can be used for modeling nonlinear dynamics and it is a combination of nonlinear autoregression, moving average and exogenous inputs.
  • Sparse identification of nonlinear dynamics (SINDy): This is a data-driven model identification method that can be used to identify the underlying dynamics of nonlinear systems from time-series data.

W2 Notes

RNN (Recurrent Neural Network)

  • Designed to work with sequential data, such as time series, text, and speech

  • Called "recurrent" because the same operation is performed for every element of a sequence, with the output being dependent on the previous computations.

  • Key difference from traditional feedforward neural networks: information persists across different time steps

    • Consists of a layer of "memory cells" that can maintain an internal state, which is passed from one time step to the next
    • Allows the network to maintain information about the previous inputs, which is useful when processing sequences of data

  • The basic building block is the recurrent unit which consists of:

    1. A hidden state that is updated at each time step.

      It is calculated as a function of the input and the previous hidden state:

      ht=f(Whh⋅ht−1+Wxh⋅xt+bh)

    where xt is the input at time t, Wxh and Whh are weight matrices, and bh is a bias vector.

    The function f is typically an activation function, such as a sigmoid or a ReLU.

    1. An output that is produced based on the current input and current input state

      It is typically produced like so:

      yt=Why⋅ht+by

      where Why is a matrix of weights that determine how the hidden state is combined with the output, and by is a bias term.

  • In practice, RNNs use different types of recurrent units such as LSTM and GRU units, which are more powerful than the basic recurrent unit.

LSTM (Long Short Term Memory)

  • Uses three gates (input, forget and output gates) to control the flow of information.

  • Each gate is implemented as a sigmoid layer that receives the input and the previous hidden state, and produces a value between 0 and 1.

  • The update equations are as follows:

    it=σ(Wxi⋅xt+Whi⋅ht−1+bi)←input gate

    ft=σ(Wxf⋅xt+Whf⋅ht−1+bf)←forget gate

    ot=σ(Wxo⋅xt+Who⋅ht−1+bo)←output gate

    ct=ft⋅ct−1+it⋅tanh⁡(Wxc⋅xt+Whc⋅ht−1+bc)←memory cell

    ht=ot⋅tanh⁡(ct)←hidden state

    where σ is the sigmoid function, defined as σ(x)=11+e−x

  • Implementation

    from keras.models import Sequential
    

    # define the model
    model = Sequential()
    # model.add(SimpleRNN(units=32, input_shape=(None, 1)))
    model.add(LSTM(units=32, input_shape=(None, 1))) 
    model.add(Dense(1))
    

    # compile the model
    model.compile(loss='mean_squared_error', optimizer='adam')

Transformer

  • A type of neural network architecture used for natural language processing tasks

  • Introduced in the paper "Attention Is All You Need" by Vaswani et al. in 2017.

  • Known for its ability to efficiently handle long-term dependencies and parallelizable computation.

  • Based on the concept of self-attention, which allows the model to weigh the importance of different parts of the input when making predictions

    • Calculates the attention score between two vectors as their dot product.

  • Uses an encoder-decoder architecture, where the encoder is responsible for processing the input sequence and the decoder is responsible for producing the output sequence

Attention mechanism

Attention can be mathematically represented as:

Attention(Q,K,V)=softmax(QKTdk)V

where Q, K, and V are matrices representing the query, key, and value respectively. dk is the dimension of the keys.

  • The attention mechanism is applied multiple times in the Transformer model, in a multi-head attention mechanism, where multiple sets of queries, keys, and values are used. The output of the multi-head attention is then concatenated and passed through a linear layer.

Implementation

xxxxxxxxxx
from keras.layers import Input, Transformer, Dense
from keras.models import Model


# define the input
input_tensor = Input(shape=(None,))


# define the transformer layer
transformer_layer = Transformer(num_heads=8, d_model=256, ff_dim=512)


# apply the transformer layer to the input
x = transformer_layer(input_tensor)


# add a dense layer for the output
output_tensor = Dense(units=1)(x)


# create the model
model = Model(input_tensor, output_tensor)


# compile the model
model.compile(optimizer='adam', loss='mean_squared_error')


# fit the model to the training data
model.fit(X_train, y_train, batch_size=32, epochs=100)
